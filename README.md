# coaching-projet-1

## Back-end

### Installation

```bash
cd back-end
yarn install
```

### Démarrage

```bash
yarn start
```

## Front-end

### Installation

```bash
cd front-end
yarn install
```

### Démarrage

```bash
yarn start
```
